How to run:
sh run.sh

How to change the input file:
open the run.sh document and change the argument on line 6 to the file you want
example:
./bin/mmm theFileIWant.txt

Input:
There are several input examples available the folder called "input"

Output:
The CFG is outputted after parsing
and a second time after the clean up
and a last time after the entire algorithm
